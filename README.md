# Shop-Cart-Art-Gallery
This project is a simple yet functional “Add to Cart” Users can browse products, add or remove items from the shopping cart, and view real-time updates of total prices. The project highlights clean code structure, responsive design, and beginner-friendly logic for anyone learning e-commerce front-end concepts.


Features:
Displays a dynamic list of products from a JavaScript array
Allows users to add, update, and remove items from the cart
Automatically updates the total price when quantities change
Uses clean, readable JavaScript code for learning and practice
Beginner-friendly design for understanding e-commerce logic

Concepts 
Dynamic HTML rendering with insertAdjacentHTML()
Array methods: forEach(), find(), filter()
Real-time UI updates based on cart state
DOM event handling for buttons and inputs

Preview
Each product displays an image, name, and price.
Click “ADD TO CART” to include it in your cart, where you can change quantities or remove items. The total price updates automatically after each change.
